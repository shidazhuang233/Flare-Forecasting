# 太阳耀斑预报

基于深度卷积神经网络的全日面太阳磁场特征建模与耀斑预报。

## 所做工作

针对全日面太阳磁场，构建了AlexNet、VGGNet、Inception和ResNet四种深度卷积神经网络，用于实现太阳耀斑预报。
